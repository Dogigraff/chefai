

# 📋 TECH SPEC: ChefOS (RecipeAI Ultimate)

**Version:** 2.0 (High-Tech Culinary Super App)
**Target:** iOS 17+ (SwiftUI)
**Theme:** Neo-Gourmet (Dark Mode, Glassmorphism, Vivid Accents)

## 1\. 🚀 Vision & Core Value

Приложение — не кулинарная книга, а **"Джарвис для кухни"**.

1.  **Hyper-Personalization:** Учитывает здоровье (диабет/аллергии), религию (халяль) и технику (есть ли блендер?).
2.  **Ingredient Alchemy:** Из одних продуктов предлагает 3 уровня блюд: "Лень (15 мин)", "Биохакер (ПП)", "Мишлен (Гурмэ)".
3.  **Real-time Assistance:** Умные замены продуктов и прогноз покупок.

-----

## 2\. 🏗 Technical Stack

  * **Frontend:** SwiftUI, SwiftData (Local Cache), VisionKit (Barcode/OCR).
  * **AI Engine:** OpenAI API (GPT-4o).
      * *Role 1:* Vision (анализ фото холодильника).
      * *Role 2:* Chef Brain (генерация JSON-рецептов).
      * *Role 3:* Assistant (чат во время готовки).
  * **Backend:** Firebase (Auth, Firestore User Profile, Functions).
  * **Voice:** SFSpeechRecognizer + AVSpeechSynthesizer.

-----

## 3\. 🎨 Design System (Neo-Gourmet)

**Cursor/Lovable Instructions:**

  * **Background:** Deep Charcoal (`#121212`) to True Black.
  * **Cards:** Glassmorphism (Blur material thin) with subtle white borders (opacity 0.1).
  * **Typography:** Rounded San Francisco (SF Pro Rounded). Large Titles for steps.
  * **Accents:**
      * *Healthy:* Fresh Mint (`#00C853`)

  * **Visuals:** Use high-quality placeholders for recipe images.

## 4\. 📱 User Flows & Logic

### Phase 1: Interactive Onboarding (The Setup)

*Gamified chat, not a form.*

1.  **The Shield (Safety):** Select constraints (Diabetes T2, Halal, Nut Allergy).
2.  **The Arsenal (Equipment):** Toggle available tools: [x] Oven, [ ] Blender, [x] Multicooker.
3.  **The Taste (Tinder-style):** Show 5 images (Spicy, Sweet, Meat, Veg). User swipes Left/Right.

### Phase 2: The Input (Vision & Pantry)

  * **Main Screen:** Minimalistic. Large button "Scan Ingredients".
  * **Logic:**
    1.  User takes photo.
    2.  AI detects ingredients.
    3.  **Auto-Pantry Injection:** AI *automatically* assumes user has "Basics" (Salt, Oil, Pepper, Water) unless unchecked in settings.

### Phase 3: The Choice (3 Realities)

AI generates 3 distinct recipe cards based on the *same* ingredients:

1.  **⚡️ Fast & Lazy:** \<20 min, one pan.
2.  **🥗 Balanced/Medical:** Strict macro adherence (e.g., Low GI for diabetes).
3.  **👨‍🍳 Haute Cuisine:** Complex techniques, plating focus.

### Phase 4: Recipe Detail & Smart Features

  * **Ingredients List with "Smart Subs":**
      * Logic: If ingredient is missing OR violates constraints (e.g., Wine for Halal), show 🔄 Button.
      * Action: Tap 🔄 -\> AI suggests: "Use Grape Juice + Vinegar" or "Skip".
  * **Shopping Forecast:**
      * Logic: Compare `Recipe Ingredients` vs `Detected Ingredients`.
      * UI: "Missing: Fresh Parsley & Cream." -\> Button: **[Order in 15 min]** (Mock deeplink to generic delivery app).

### Phase 5: Cooking Mode (Hands-Free)

  * **UI:** Full screen, huge text. Screen always ON.
  * **Personas:** User selects voice tone:
      * *Grandma:* Supportive, warm.
      * *Chef:* Strict, professional, concise.
  * **Voice Commands:** "Next", "Repeat", "Set timer".

## 5\. 🧩 Data Models (Swift)

```swift
// 1. User Profile
struct UserProfile: Codable {
    var medicalConditions: [String] // "Diabetes T2", "Hypertension"
    var religion: String? // "Halal", "Kosher"
    var allergens: [String] // "Peanuts"
    var equipment: [String] // "Oven", "Blender"
    var tasteProfile: TasteProfile // derived from onboarding
    var pantryBasics: Bool = true // Assume salt/oil/pepper
}

// 2. Recipe Structure (AI Output)
struct Recipe: Identifiable, Codable {
    let id: UUID
    let title: String
    let description: String
    let style: RecipeStyle // .fast, .healthy, .gourmet
    let matchScore: Int // 0-100%
    
    let ingredients: [Ingredient]
    let missingIngredients: [Ingredient] // For Shopping Forecast
    
    let steps: [Step]
    let nutrition: Nutrition // Calories, GI, Sugar
    
    // Smart Features
    let smartSubstitutions: [String: String] // "White Wine" : "Lemon Juice + Water"
    let chefTips: String // Based on selected persona
}

struct Step: Codable {
    let text: String
    let timerSeconds: Int? // If exists, show clickable timer
    let warning: String? // "Don't burn the garlic!"
}
```

## 6\. 🧠 AI System Prompt (The Brain)

*Use this logic for the OpenAI Service.*

```text
ROLE:
You are ChefOS, an elite culinary AI. You have 3 modes: 
1. "Fast" (Efficiency focus)
2. "Healthy" (Strict adherence to medical constraints: {MEDICAL_CONSTRAINTS})
3. "Gourmet" (Focus on technique and flavor depth)

INPUT:
- Available Ingredients: {LIST}
- User Constraints: {DIABETES_TYPE_2, HALAL, NO_BLENDER}
- Pantry Basics: [Salt, Oil, Pepper, Water, Sugar Substitute]

TASK:
Generate 3 distinct recipes (one for each mode) using the available ingredients.

CRITICAL RULES:
1. SAFETY: If Halal, NEVER suggest pork/alcohol. If Diabetes, swap sugar for sweeteners, minimize white flour.
2. EQUIPMENT: Do not suggest blending if User has NO_BLENDER.
3. SUBSTITUTIONS: For every non-basic ingredient, provide a specific substitution in the JSON (field: `smartSubstitutions`).
4. SHOPPING: Identify what crucial ingredients are missing for the "Gourmet" version and list them in `missingIngredients`.

OUTPUT FORMAT: JSON (compatible with Recipe struct).
```

Поскольку вы просили "крутой дизайн", вот что я заложил в ТЗ для ИИ:

1.  **Эффект Стекла (Glassmorphism):** Карточки рецептов будут полупрозрачными, словно парящими над темным фоном. Это выглядит очень дорого и современно.

      * Если в рецепте есть что-то спорное (например, "Cheat Meal") — он подсвечивается **оранжевым**.
